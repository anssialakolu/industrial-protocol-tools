# Industrial Protocol Tools

A collection of Python tools for analyzing and interacting with industrial communication protocols used in factory automation and PLC systems.

The repository contains small projects focused on learning and exploring common industrial protocols.

## Projects

- **Modbus TCP Master** – communicate with Modbus TCP devices using Python sockets
- **PROFIBUS-DP Telegram Decoder** - decode telegrams from raw bitstreams
- **Possible future projects** – PROFINET, EtherCAT, EtherNet/IP, OPC UA and CAN bus.

## Purpose

The goal of this repository is to better understand industrial communication protocols by implementing simplified tools for decoding, device discovery and data acquisition etc.

## Technologies

- Python
- Socket programming
- Wireshark & PCAP analysis
- Signal analysis
- Industrial communication protocols
